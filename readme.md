# 穷舍怪人游戏通关指南

## 游戏简介
《穷舍怪人》是一款冒险解谜游戏。玩家控制主角穷舍怪人在一个神秘岛屿上探索，收集物品，与NPC对话，最终逃离即将毁灭的世界。

## 通关步骤详解

### 步骤1: 开始游戏
- 点击"点击开始"进入游戏
- 查看故事介绍：穷舍怪人降落到新岛屿，需要弄清楚位置

### 步骤2: 探索农场
- 移动到农场场景
- **攻击树**：点击树木，使用"攻击"动作，掉落苹果
- **收集苹果**：使用"拾取"动作收集苹果
- **收集紫罗兰**：在农场右下角找到紫罗兰花，使用"拾取"收集
- **收集鸭子**：靠近农场中间的鸭子，使用"拾取"。如果距离太远，鸭子会逃跑，需要多次尝试靠近才能成功收集

### 步骤3: 探索房子
- 移动到房子场景
- **与老人对话**：点击老人，使用"对话"，第一次会被赶出去
- **给老人紫罗兰**：使用"使用"动作，选择紫罗兰给老人
- 老人会接受花并给你梯子
- **收集物品**：
  - 镜子（老人旁边）
  - 卷轴（桌子上，可阅读了解世界末日预言）
  - 玫瑰花（角落）

### 步骤4: 探索屋顶
- 从房子进入屋顶
- **收集铲子**：屋顶上有铲子，使用"拾取"收集

### 步骤5: 探索洞穴
- 回到山脉，进入洞穴（需要火把点亮，但现在还不需要）
- **挖石头**：使用铲子"使用"石头，挖20次打开通路
- 进入洞穴深处查看古老涂鸦

### 步骤6: 探索地下洞穴
- 从山脉进入地下洞穴
- **吓唬蜘蛛**：使用镜子"使用"蜘蛛，蜘蛛会被吓跑
- 进入更深的洞穴收集火把

### 步骤7: 返回山脉准备逃离
- 回到山脉场景
- **收集鸭子**：如果还没收集完，继续靠近鸭子使用"拾取"直到收集（鸭子会逃跑，需要靠近才能成功）
- **上月球**：使用梯子"使用"月亮，爬上月球

### 步骤8: 月球结局
- 与coso对话
- 拯救鸭子
- 观看结局动画

## 游戏机制说明

### 动作系统
- **查看**：观察物品或人物
- **对话**：与NPC交谈
- **拾取**：收集物品
- **攻击**：破坏物品或战斗
- **阅读**：阅读文字内容
- **使用**：使用物品

### 重要物品
- **苹果**：喂给coso让他跳上月球
- **紫罗兰**：给老人换取梯子
- **梯子**：上月球的工具
- **铲子**：挖开洞穴道路
- **镜子**：吓唬蜘蛛
- **火把**：照亮黑暗区域
- **鸭子**：coso的宠物，必须拯救

### NPC角色
- **coso**：本地人，需要食物才能帮助
- **老人(ao)**：房子主人，知识渊博
- **鸭子**：coso的宠物

## 提示
- 有些物品需要多次交互才能收集
- 对话顺序很重要，有些NPC的对话会解锁新内容
- 仔细探索每个场景，不要遗漏物品
- 世界末日预言暗示了游戏背景
- **鸭子收集技巧**：鸭子很胆小，如果距离太远会逃跑。需要靠近鸭子（角色与鸭子重叠）才能成功拾取

## 技术信息
- 游戏基于HTML5 Canvas开发
- 支持触摸和鼠标操作
- 完全汉化版本
